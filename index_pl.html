O mnie

Jestem freelancerem specjalizującym się w web scrapingu, z ponad 2-letnim praktycznym doświadczeniem w tworzeniu niestandardowych narzędzi do ekstrakcji danych. Przez ostatni rok skupiałem się również na rozwijaniu rozwiązań automatyzacyjnych i usprawnianiu przepływów pracy związanych z przetwarzaniem dokumentów.

Moje podejście łączy umiejętności techniczne z naciskiem na praktyczne rezultaty, mając na celu dostarczanie wydajnych rozwiązań, które oszczędzają czas, redukują wysiłek manualny i dostarczają cennych informacji z danych.

Doświadczenie

Kluczowe umiejętności i obszary, w których dostarczam rozwiązania.

Web Scraping

Tworzenie niestandardowych scraperów (Python, Selenium, Requests) do wydajnego i niezawodnego pozyskiwania danych ze stron internetowych.

Automatyzacja Zadań

Tworzenie skryptów i botów do automatyzacji powtarzalnych zadań, wprowadzania danych, generowania raportów i nie tylko.

Przetwarzanie Dokumentów

Ekstrakcja ustrukturyzowanych danych z plików PDF i zeskanowanych dokumentów przy użyciu narzędzi takich jak Docparser i niestandardowych skryptów.

Ekstrakcja i Czyszczenie Danych

Przekształcanie surowych, zescrapowanych lub wyekstrahowanych danych w czyste, zorganizowane i użyteczne formaty (CSV, JSON, Baza danych).

Optymalizacja Procesów

Analiza manualnych przepływów pracy w celu identyfikacji możliwości automatyzacji i poprawy wydajności.

Integracja API

Łączenie różnych systemów i usług za pomocą API w celu zapewnienia płynnego przepływu danych i zautomatyzowanych działań.

Wyróżnione Projekty

Przykłady rozwiązań, które zbudowałem.

Scraper Użytkowników Twittera

Opracowałem skrypt w Pythonie do scrapowania najnowszych tweetów z określonych profili użytkowników. Wykorzystałem Selenium do początkowego logowania i pobierania plików cookie, a następnie wykonywałem bezpośrednie żądania w celu efektywnego pobierania danych, omijając ograniczenia interfejsu użytkownika.

Python Selenium Requests Web Scraping

Rezultat: Dostarczyłem klientom aktualne strumienie danych z kluczowych kont Twittera do analizy i monitorowania.

Przetwarzanie Dokumentów z Docparser

Zautomatyzowałem ekstrakcję ustrukturyzowanych danych z różnych typów dokumentów (PDF, zeskanowane faktury, formularze) przy użyciu platformy Docparser i jej API. Skonfigurowałem reguły parsowania i zintegrowałem wyekstrahowane dane z bazami danych lub arkuszami kalkulacyjnymi klientów.

Docparser Integracja API Python Automatyzacja Ekstrakcja Danych

Rezultat: Znacząco zredukowałem czas ręcznego wprowadzania danych o ponad 90% i poprawiłem dokładność danych przy przetwarzaniu faktur.
